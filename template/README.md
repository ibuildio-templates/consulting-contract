## Install

To install latex on linux,

```bash
sudo apt-get install texlive
```

On OS X, install [MacPorts](http://www.macports.org/) first then,

```bash
sudo port install texlive
```

Install npm modules

```bash
npm install
```


## Build

``` bash
npm run build
```
